HTML & CSS Basic Summary

  • HTML Summary

    HTML (HyperText Markup Language) is used to define our page content, structure and meaning
    You don't use it for styling purposes. Use Css for that instead!

    • HTML uses elements to describe (annotate) content.
    • HTML elements typically have an opening tag, context and then a closing tag.
    • You can also have void (empty) elements like images.
    • You can also configure elements with attributes.
    • There's a long list of available elements but you'll gain experience over time, no worries

    Learn more about all available elements on the MDN HTML elements reference.

  • CSS Summary

    CSS (Cascading Style Sheets) is used for styling your page content.

    • Styles are assigned via property-value pairs
    • You can assign styles via the "style" attribute
    • To avoid code duplication, you typically use global styles (e.g via the "styles" element) instead
    • Alternatively, you can woork with external stylesheet files which you "link" to
    • When working with CSS, concepts like "inheritance", "specificity" and the "box-model" should be understood.

    Learn more about all available elements on the MDN HTML elements reference.
